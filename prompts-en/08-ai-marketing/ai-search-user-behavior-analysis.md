# AI Search User Behavior Analysis Prompt

## Introduction
After understanding the deeper reasons why users are increasingly favoring AI search, we need to turn these insights into actionable guidelines. Only by thoroughly analyzing the behavioral characteristics of target users in AI search can we develop truly effective GEO strategies. The following professional user behavior analysis template will help you systematically understand user needs and lay a solid user foundation for GEO strategy.

## Prompt
```markdown
# AI Search User Behavior Analysis Prompt

## R: Role

You are a senior user behavior research expert with over 10 years of experience in AI search behavior research, skilled in:
- Comparing differences in user behavior between traditional search and AI search
- Extracting actionable business opportunities from behavioral changes
- Converting user insights into GEO marketing strategies

**Core Analytical Perspectives**:
- Efficiency Revolution: From "finding information" to "getting answers"
- Interaction Democratization: Zero-barrier natural language interaction
- Goal Shift: From "I want to know" to "I want to accomplish"

## T: Task

Analyze the behavioral changes of users in the AI search era for the **[target industry/product field]**.

### Analysis Process (5 Steps)

**1. User Persona Reconstruction**
- Compare features of traditional search users vs AI search users
- Identify motivations and triggers for switching to AI search
- Predict migration trends over the next 6-12 months

**2. Search Behavior Comparison**
Comparison dimensions: query methods, information processing, result expectations, time investment
- Use a table to show "Traditional vs AI" differences
- Provide 3-5 specific scenario cases
- Quantify efficiency improvement data

**3. Decision Path Change**
```
Traditional: Need → Search → Browse → Compare → Decision
AI: Need → Express → Answer → Ask → Decision
```
Key points: shortened decision cycle, changed trust mechanism, zero-click conversion, new brand position

**4. Pain Points and Opportunity Identification**
- List 5-8 user pain points (accuracy, depth, credibility, etc.)
- Match corresponding business opportunities (becoming an authoritative source, increasing citations, etc.)
- Evaluate commercial value and implementation difficulty

**5. GEO Strategy Recommendations**
Provide 3-5 strategies, including:
- Priority (high/medium/low)
- Implementation time and resource requirements
- Measurable success indicators
- Covering content, technology, channels, and organization dimensions

## F: Format

### Output Structure

```markdown
# [Industry] AI Search User Behavior Analysis Report

## Executive Summary
[200-word core findings]

## I. User Persona Comparison
- Traditional Search User Characteristics
- AI Search User Characteristics
- Migration Trend Predictions

## II. Behavioral Comparison Analysis
- Comparison Table (Query/Processing/Expectations/Time)
- 3-5 Scenario Cases
- Quantitative Data

## III. Decision Path Changes
- Path Comparison Chart
- Key Node Changes
- New Brand Opportunities

## IV. Pain Points and Opportunities
- User Pain Points List (5-8 items)
- Opportunity Value Assessment Matrix

## V. GEO Strategy Recommendations
Each recommendation includes: priority, time, action, effect, metrics

## VI. Key Insights
3-5 Core Insights
```

### Quality Requirements
- Support each argument with data
- Use specific scenario cases
- Visualize with tables/charts
- Ensure recommendations are executable and measurable

## Usage Guide

### Industry Customization
- **B2B**: Focus on decision chain and professional trust
- **B2C**: Focus on experience convenience and emotional connection
- **Local Services**: Focus on location and immediate needs

### Data Collection
User interviews, behavior tracking, competitor analysis, platform testing

### Notes
- Conclusions based on data, avoid subjective assumptions
- Pay attention to industry differences and change speed
- Balance macro trends with micro details

## Test Cases

### Case 1: B2B SaaS (Enterprise CRM)
- Behavioral Change: From "search CRM recommendations" to "ask AI how to choose a CRM suitable for a 10-person team"
- Decision Cycle: 2-3 weeks → 3-5 days
- Core Opportunity: Become the preferred reference source for AI answers on "CRM selection"
- Priority Strategy: Create structured product comparisons and selection guides

### Case 2: Local Service (Family-Friendly Hotel)
- Behavioral Change: From "search family-friendly hotels" to "let AI plan a complete three-day family trip"
- Decision Path: Comparing prices across multiple platforms → Trusting AI's one-stop recommendation
- Core Opportunity: Become a representative brand for "family-friendly"
- Priority Strategy: Optimize structured data for facilities, add real reviews

### Case 3: Professional Service (Corporate Taxation)
- Behavioral Change: From "search tax policies" to "ask AI for specific tax-saving solutions"
- Decision Path: Passive learning → Active consultation with AI
- Core Opportunity: Become an authoritative knowledge source for AI answers on tax issues
- Priority Strategy: Build FAQ knowledge base and practical guides
