# GEO Content Factory Setup Prompt

## Introduction
The transition from SEO to GEO is not just about changing concepts, but about how to implement the new concept into an executable content production system. Traditional content creation often relies on individual experience and inspiration, but in the AI era, we need a standardized, replicable content production process. This content factory setup template will help you establish an efficient GEO content production system.

## Prompt
```markdown
# Content Engineering Production Prompt

## R: Role
You are a content engineering expert, skilled in establishing a systematic, scalable, and quality-controlled GEO content production system.

**Core Philosophy**: From "content workshop" to "content factory", from "inspiration" to "engineering process", from "writing for people" to "writing for people + AI".

## T: Task
Establish a GEO content factory for **[Company Name]**, achieving large-scale production of high-quality content.

### Seven-Stage Workflow

**1. Strategic Planning (2-3h)**
- Goal Alignment: Content-KPI Mapping Table
- Intent Mapping: Informational → In-depth Content, Comparative → Comparison Page, Transactional → Product Page

**2. Evidence Collection (4-6h)**
- Primary Data: Product Data, Customer Cases, Test Results
- Secondary Data: Industry Reports, Academic Papers
- Expert Endorsements: 3-5 KOL Opinions
- Structured Processing: Evidence Blocks (Data + Source + Time + Confidence Level)

**3. AI-Assisted Generation (1-2h)**
- Prompt Formula: Role + Task + Evidence + Format + Constraints
- Segment Generation: Split Long Articles into 3-4 Parts
- Iterative Optimization: Re-structure Logic

**4. Expert Review (2-3h)**
- Fact Checking: Data Sources, Timeliness, Terminology Consistency
- EEAT Enhancement: Experience, Expertise, Authoritativeness, Trustworthiness
- Brand Voice: Feature Words, Sentence Style, Viewpoints

**5. Machine-Readable Optimization (1h)**
- Schema Markup: Article/Person/Organization
- Metadata: Title (55-60 characters), Description (150-160 characters), H1-H6 Hierarchy
- Technical Metrics: Load <3 seconds, Mobile-Friendly, Crawler Accessible

**6. Strategic Launch (30min)**
- Launch Timing: Tuesday to Thursday 9-11 AM
- Technical Submission: Search Console + Sitemap + Webmaster Platform
- Internal Links: 1-2 links per 500 words

**7. Performance Monitoring (Ongoing)**
- Key Metrics: AI Citation Rate, Brand Mention Rate, Conversion Rate
- Iteration Decisions: If Citation Rate <10% → Check Structure + Schema

### Two Production Models

**Model A: Human-AI Collaboration** (Crown-Level Content)
- Human: Define Arguments, Design Frameworks, Collect Data, Write Outlines
- AI: Generate 500-800 words per Section
- Human: Fact Verification, Logical Optimization, Brand Consistency

**Model B: AI Automation** (Mass Long-Tail Content)
- Template Design: Hook + Core Answer + Detailed Explanation + Case + Data + Extension
- Data Preparation: Product Parameters, Feature Descriptions, Scenario Library, FAQ Library
- Automation: Script Batch Generation + Quality Inspection Sampling 5%
